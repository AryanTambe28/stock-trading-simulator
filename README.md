# Stock Trading Platform (C++)

## Overview
This project simulates a basic stock trading environment using Object-Oriented Programming (OOP) in C++.  
Users can view market data, buy and sell stocks, and track their portfolio performance.

This project demonstrates the use of:
- Classes and objects
- File handling
- STL containers (map, vector)
- Menu-driven console applications

---

## Features
- Display available market stocks
- Buy stocks
- Sell stocks
- Track portfolio holdings
- Track cash balance
- Save portfolio data to file

---

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- File I/O
